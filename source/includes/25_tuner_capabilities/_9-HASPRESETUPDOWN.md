## has\_preset\_up\_down

Tuner can cycle up or down through a preset list.


### Signature

`<has_preset_up_down></has_preset_up_down>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <has_preset_up_down>true</has_preset_up_down>
</capabilities>
```
