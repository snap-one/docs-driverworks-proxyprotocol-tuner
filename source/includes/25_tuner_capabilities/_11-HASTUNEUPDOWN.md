## has\_tune\_ up\_down

Tuner can cycle up or down through tuning frequencies or channels


### Signature

`<has_tune_up_down></has_tune_up_down>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <has_tune_up_down>true</has_tune_up_down>
</capabilities>
```
